X-plane-teensy-ufcp

Arduino source code, PCB layouts and full documentation on a Teensy++ based Autopilot panel for fighters in X-Plane 10.30+

Download this project as a .zip file Download this project as a tar.gz file

Project uses a single Teensy 2.0++ controller to imitate the advanced autopilot options found in fighter jets.
